App protection with AWS

Express.js app written as AWS (Amazon Web Services) Lambda function that gives access to your app only to authorized users. All user logs (POST requests) are saved in PostgreSQL database (Amazon RDS). Users can be manually blocked. Lambda is triggered with AWS API Gateway. Plus, additional offline protection.

AWS Lambda sends email with statistics every week using nodemailer library. Email service is protected with google OAuth2. Lambda is triggered with a Rule (Cron expression) in EventBridge (CloudWatch Event). Default 3s Lambda timeout is increased in case of long requests. Sensitive info (keys) stored as Lambda environment variables.

Online store

React online store. Navigation is implemented with ReactRouter. Authentication, registration, database are processed with Firebase cloud service. User info, products and shopping cart are managed with Redux, Reselect and stored with Redux-Persist. Components are styled with SASS and Styled-components library. Notification component is managed with React Context. Deployed on Netlify.

React   JSX   Firebase   SASS   StyledComponents   ReactRouter   Redux   Reselect   Redux-Toolkit   Redux-Persist   Redux-Thunk   Redux-Saga   Netlify

Carousel

Slideshow made with Javascript, CSS and HTML. Autoplay is set to 3s. When the slide is hovered, autoplay stops. When a cursor leaves the carousel, the autoplay continues. Buttons are processed with a 'throttle' function to allow the animation to stop before you can move to the next slide.
